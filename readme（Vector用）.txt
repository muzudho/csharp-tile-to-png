ソフトの概要
============

    タイル状に画像を並べて結合し、１枚の画像として出力します。
    カスタマイズしたい場合は  C#ソースコード  をいじってください。

────────────────────────────────────────
チュートリアル
────────────────────────────────────────


（１）

    TileToPng.exe をダブルクリックして、アプリケーションを起動してください。

（２）
    ウィンドウサイズが小さいので、ウィンドウの右下の角をドラッグ＆ドロップ
    するなどして、ウィンドウのタテ、ヨコを広げてください。

（３）
    キーボードの [Ctrl] + [L] ボタンを押してください。
        （[Ctrl]ボタンを押しながら[L]ボタンを１回だけ押して、すぐに離してください）

    画像を読込みます。しばらく待ってください。

（４）
    囲碁のような画像（サンプル画像）が画面上にセットされると思います。

（５）
    キーボードの [P] ボタンを押してください。

    .exe と同じフォルダー内に 画像を出力します。
    例：TileToPng_20160718_201042_539.png

    このように画像を作るのが、このアプリケーションの機能です。

（６）
    画面上は、５つのレイヤー（階層）が重なって表示されています。
    画面左上の５つの○印をクリックすることで、レイヤーを変えることができます。

（７）
    青色の四角がカーソルです。
    マウスで画面上をクリックしたり、キーボードのカーソルキー（↑→↓←）を押してみてください。
	カーソルを見やすい位置にずらしてください。

（８）
    添付の「サンプル画像」フォルダーの中の .png 画像を、
    ウィンドウ上にドラッグ＆ドロップしてください。

    青い四角のカーソルを左上の原点として、画像が配置されます。

（９）
    金色の罫線（グリッド）の右下にある  ○印  をドラッグ＆ドロップして、
    画像として出力したいサイズを決めてください。

（１０）
    作業を途中で中断する場合は、[Ctrl]+[S]キーで保存してください。
    再開するときは [Ctrl]+[L]キーで再開してください。

（１１）
    そのほかの説明は、下記を参照してください。


────────────────────────────────────────
画面の見方
────────────────────────────────────────

    縦に５つ並んでいる○
    
        画像レイヤー

    罫線

        ドラッグ＆ドロップで
        画像を置いていく。

    青い四角

        カーソル。
        画像を置く位置を指している。

    罫線右下の○

        リサイザー。
        ドラッグ＆ドロップで
        出力時の画像サイズを指定。


────────────────────────────────────────
操作方法
────────────────────────────────────────

    画像出力

        [P]キーで画像出力。（.exeと同じフォルダーに出力）

        グリッド右下の○印をドラッグ＆ドロップで、出力時の画像サイズを縮めることができる。
        最大サイズは  zレイヤー : yタテ : xヨコ = 5x30x30 枚固定


    画像配置

        [↑][→][↓][←]キーで  青い四角のカーソル移動。

        ドラッグ＆ドロップで画像を読込み。
        グリッドは 40x40ピクセル 固定だが、
        40x40ピクセル以外の画像も読込める。

        [Ctrl]+[C] コピー
        [Ctrl]+[X] カット
        [Ctrl]+[V] ペースト

    レイヤー変更

        レイヤーの○をクリックすることで、画像を置く階層を変更可能。

    セーブ／ロード

        現在作業中の状態を、簡易的に保存しています。（.exeと同じフォルダーに出力）

        [Ctrl]+[S] セーブ（保存）
        [Ctrl]+[L] ロード（再開）

        .exe と同じフォルダーの下にある画像であれば、
            そのフォルダーからの相対パスで保存するので、
            フォルダーを丸ごと移動することで  環境を移動することができます。
        そうでなければ絶対パスで保存します。
